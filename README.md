Relógio Digital Simples

Este é um projeto simples de um relógio digital criado usando HTML, CSS e JavaScript. O relógio exibe a hora atual em formato de 24 horas e é atualizado automaticamente a cada segundo.

Funcionalidades
Exibe a hora atual em tempo real.
Formato de 24 horas.
Atualização automática a cada segundo.
Tecnologias Utilizadas
HTML
CSS
JavaScript
